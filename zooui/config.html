class zooui.config.ConfigManager(config_file=None)[source]¶

Bases: object

Manages ZooUI configuration with ~/.zooui/config.json as single source of truth.

Features: - Creates default config file if missing - Validates configuration against schema - Rejects invalid values with clear errors - Supports temporary overrides via –config

Initialize ConfigManager.

config_file – Optional custom config file path (for testing)
