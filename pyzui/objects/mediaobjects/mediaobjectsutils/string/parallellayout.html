Manages parallel calculation of text layouts using thread pools.

This class provides thread-safe parallel calculation of text layouts for batches of objects, with progress tracking and error handling.

max_workers¶

Maximum number of worker threads

batch_timeout_ms¶

Timeout for batch calculations in milliseconds

Whether to enable performance profiling
