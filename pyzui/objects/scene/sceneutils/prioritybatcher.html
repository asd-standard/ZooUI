pyzui.objects.scene.sceneutils.prioritybatcher module¶

PriorityBatcher class for viewport-aware object batching.

This class manages batching of text objects for parallel rendering based on their distance from the viewport center, ensuring closest objects are rendered first.

class pyzui.objects.scene.sceneutils.prioritybatcher.PriorityBatcher(batch_size=10, max_batches=10, viewport_center=None, viewport_rect=None)[source]¶

Bases: object

Manages batching of text objects for parallel rendering.

This class organizes text objects into priority batches based on their distance from the viewport center, ensuring that objects closest to the viewport are rendered first for optimal perceived performance.

Number of objects per batch

max_batches¶

Maximum number of batches to create

Current viewport rectangle
